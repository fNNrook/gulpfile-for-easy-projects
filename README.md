# gulpfile-for-easy-projects
An easy gulpfile, for little projects and doings

it contains: 
* browser-sync
* gulp-autoprefixer
* gulp-concat
* gulp-html
* gulp-htmlmin
* gulp-sass
* gulp-sourcemaps

## installation:
`npm i` for install the dependencies

## orders:
Use `npm run build` to build your Production Folder

Use `npm run start` to start your watch task
